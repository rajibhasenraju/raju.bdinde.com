ABOUT THE COURSE

This Course training on Certificate in Database Programming provides a solid foundation in the basic and intermediate skills for working with Relational Database Management System (RDBMS), MS SQL Server and Database Connectivity with Visual Basic.NET. This course is ready to extend the knowledge into some of the more specialized and advanced capabilities of software development.


WHAT I LEARNED FROM THIS COURSE:

• Introduction to Database Systems.
• Database Concepts and Structure.
• Normalization as a Process for Verification of Data Model Design.
• The Relational Algebra.
• SQL Query and Data Manipulation.
• SQL Functions.
• Defining Database Object Structure.
• User Security and Error Handling.
• Database Backup and Recovery.
• SQL Interaction with Programming Interfaces.
• Desktop Database Packages.
• Data Quality and Database Administration.
• Introduction to Data Warehousing.
• Overview of Object‐Oriented Databases.
• Practical Project.
